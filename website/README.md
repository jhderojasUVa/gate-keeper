# Gate Keeper Website

Official website for Gate Keeper - Your first line of defense in trunk development.

Built with **React 18** and **TypeScript** for type safety and better developer experience.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Preview

```bash
npm run preview
```

## Tech Stack

- React 18
- TypeScript
- Vite 6
- CSS3
